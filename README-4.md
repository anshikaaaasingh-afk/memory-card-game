# Ultimate Memory Card Game 🧠🃏

A fun, interactive Memory Card (Match the Pairs) game built with vanilla
HTML, CSS, and JavaScript.

## About

Flip cards, find matching pairs, and test your memory! This project focuses
on clean UI, smooth animations, and engaging gameplay — no frameworks or
libraries required.

## Features

- Card flip animations with smooth CSS transitions
- Randomized card shuffle every game
- Move counter and timer to track performance
- Match detection with visual feedback for correct/incorrect pairs
- Win screen once all pairs are found
- Responsive design — playable on desktop and mobile
- Restart/New Game button to reset instantly

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, Safari)

### Run Locally

1. Clone or download this repository.
2. Open `index.html` directly in your browser, **or**
3. Use VS Code's **Live Server** extension for auto-reload during development.

```bash
git clone https://github.com/anshikaaaasingh-afk/memory-card-game.git
cd memory-card-game
```

Then open `index.html` in your browser.

## How to Play

1. Click any card to flip it and reveal what's underneath.
2. Click a second card to try to find its match.
3. If the two cards match, they stay flipped.
4. If they don't match, both cards flip back after a short delay.
5. Match all pairs in the fewest moves and shortest time to win!
